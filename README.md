# NConsole 1.0.0

NConsole is an in-game command console wich help you to easily debug your game.

### Features : 
* Display all Unity's logs.
* Enter Commands.
* Add your own commands.
* Autocompletion.
* Command history.
* Theme Customization.

### Commands : 
* Enum support
* Static methods
* Non-static methods
* Asynchronous methods
* Multiple execution types.
